# 界面总览

本章带您认识 .NET 桌面版应用的主界面，了解每个区域的作用和入口。

## 主窗口布局

```
┌──────────────────────────────────────────────────────────────────┐
│  ☰  海大终端           当前身份: 我    [切换身份]  [⚙ 设置]         │  ← 标题栏
├────────────┬─────────────────────────────────────────────────────┤
│            │                                                     │
│  📊 首页    │                                                     │
│  💰 账单    │           主内容区 (Tab 内容)                        │
│  📈 统计    │                                                     │
│  ⚙ 设置    │                                                     │
│            │                                                     │
│            │                                                     │
│  ─────     │                                                     │
│  🔄 同步    │                                                     │
│  👤 身份    │                                                     │
│            │                                                     │
└────────────┴─────────────────────────────────────────────────────┘

```

## 侧边导航

侧边栏是主入口，固定包含 4 个 Tab 和 2 个工具按钮：

| 项 | 作用 | 快捷键 |
|---|---|---|
| **首页** | 总览卡片：今日/本月消费、余额、笔数 | `Ctrl+1` |
| **账单** | 交易流水列表 + 筛选/搜索 | `Ctrl+2` |
| **统计** | 分类饼图、趋势、商户排行、月度对比 | `Ctrl+3` |
| **设置** | 主题、字体、验证码模式、加密、快照 | `Ctrl+4` |
| **同步** | 触发增量/全量同步的弹窗 | `Ctrl+S` |
| **身份** | 身份与账号管理 | `Ctrl+I` |

## 标题栏

- **应用名**：左侧固定
- **当前身份** + 切换按钮：右上角，点击可在多个本地身份间切换
- **设置入口**（齿轮）：快速打开设置 Tab

## 首页 Tab

- 4 张摘要卡片（今日消费、本月消费、账户余额、本月笔数）
- 最近 5 笔交易
- 一键同步按钮

## 账单 Tab

- 顶部筛选条：日期范围、金额范围、分类、商户
- 搜索框（按交易号、描述、商户）
- DataGrid：交易时间、商户、分类、金额、余额、状态
- 右键菜单：查看详情、纠正分类、删除（仅限未同步条目）

## 统计 Tab

- 左侧图表选择：分类 / 趋势 / 商户 / 月度
- 右侧详细面板：图表 + Top 10 列表
- 时间范围切换：今日 / 本周 / 本月 / 本季 / 本年 / 自定义

## 设置 Tab

按分组折叠：

- **外观**：主题、字体大小、动画
- **同步**：默认模式、并发数、早停阈值
- **验证码**：手动 / 远程 / 本地 ONNX
- **数据**：备份路径、快照、导入/导出
- **安全**：修改启动密码、加密状态
- **日志**：日志级别、查看日志文件

## 状态栏

底部一行状态信息：

- **同步状态**：空闲 / 同步中（带进度条）
- **最后同步时间**：账号 + 时间
- **应用版本号**：点击可查看更新日志

## 弹窗

触发弹窗的入口：

- **同步弹窗**（Ctrl+S）：选择账号、模式、验证码模式
- **同步详情弹窗**：实时显示进度、当前页、新增数
- **身份管理弹窗**（Ctrl+I）：身份列表、账号列表
- **新建/编辑身份弹窗**
- **新建/编辑账号弹窗**
- **快照管理弹窗**：列表、创建、恢复、删除
- **导入/导出向导**：CSV/JSON/钱迹格式

## 主题与外观

应用支持 Light / Dark / 跟随系统三种主题，可在 **设置 → 外观** 切换。字体大小支持小/中/大/特大四档。整体配色与 Tauri 版保持一致，确保跨平台体验统一。
